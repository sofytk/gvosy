**Gvosy** is is a mobile application with an AI assistant designed to help users capture, structure, and develop their thoughts without losing ideas.
The app allows users to record voice notes and freely talk about their thoughts. The AI assistant processes these recordings, structures the ideas into meaningful notes, suggests improvements, creates task lists, and helps users navigate their existing knowledge.

*Gvosy acts as a personal thinking assistant, turning unstructured thoughts into clear, actionable insights.*
##### Features
- Voice input for capturing thoughts and ideas
- AI-powered structuring of notes and ideas
- Automatic creation of task lists from thoughts 
- Intelligent search across existing notes
- Context-aware suggestions and insights
- Offline-first data storage with synchronization

##### Tech Stack
- Kotlin 
- Room (local database) 
- Koin (dependency injection) 
- Ktor (client-server communication) 
- AI integration (text processing, structuring, search)
